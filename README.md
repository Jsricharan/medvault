# 🏥 MedVault — Healthcare Management System

Full stack healthcare platform built with
Spring Boot and React.

## 📁 Project Structure

medvault/
├── backend/    → Spring Boot REST API
└── frontend/   → React Web Application

## 🚀 Quick Start

### Backend
cd backend
mvn spring-boot:run

### Frontend
cd frontend
npm install
npm start
